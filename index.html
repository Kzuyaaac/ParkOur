ParkOur

Platform: PC (Desktop)

Engine: Unreal Engine

Developer: Solo Developer

Description:

ParkOur is a parkour game developed using Unreal Engine by a solo indie developer. The game features high-quality graphics and dynamic gameplay, offering players a thrilling free-running experience. ParkOur was developed during June and uses high-quality assets from Unreal Engine's asset store.

Assets Used:

  • Motion Matching
  • Science Fiction Valley Town

How to Play:

In ParkOur, the player is given an initial 10 seconds to complete a level. As the player progresses, they can pass through checkpoints. Each time a checkpoint is passed, an additional 10 seconds is added to the timer. The goal is to complete the parkour course as smoothly and efficiently as possible, while racing against the constantly decreasing time.

Game Keybinds:

Action Key Description
Move Forward W Run forward
Move Left A Run left
Move Backward S Run backward
Move Right D Run right
Jump Space Jump upward
