Park Our - A Parkour Game

Platform: PC

Engine: Unreal Engine

Developer:

  • Indie Developer

Description:

Park Our is an exciting parkour game where you navigate dynamic obstacle courses. Time is always ticking as you challenge yourself with complex jumps, shortcuts, and thrilling parkour challenges, all within the breathtaking graphics powered by Unreal Engine.

Key Features:

  • Dynamic Movement: Perform thrilling parkour stunts across challenging courses.
  • Time-Based Gameplay: The clock is always tickingâ€”get to the finish line as fast as you can.
  • Challenging Obstacles: Test your skills with tricky jumps, shortcuts, and more.
  • Realistic Graphics: Powered by Unreal Engine, the game features stunning visual realism.

Development Story:

This game was created by me, an indie developer, using assets from the Unreal Marketplace. I completed it in one week while also preparing for the Gelar Karya Pembelajaran (GKP) event at my school. It was a challenging but rewarding project!

Assets Used:

  • Motion Matching
  • Science Fiction Valley Town

Gameplay Instructions:

Navigate the parkour course using dynamic movement. The clock is ticking, so hurry up! Use shortcuts to gain an advantage and reach the finish line as quickly as possible.

Game Keybinds:

Action Key Description
Jump Spacebar Press to perform a jump in mid-air
Move Forward W Press to move forward
Move Left A Press to move left
Move Backward S Run backward
Move Right D Press to move right
